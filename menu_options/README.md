# IBLU KALIGPT Menu Options Directory

This directory contains all menu options and features organized by category.

## Main Menu Options

1. **IBLU KALIGPT Chat** (`chat_mode/`)
   - Multi-AI provider chat interface
   - OpenAI, Gemini, and local model support
   - Interactive chat with history

2. **Hacking Toys** (`tool_management/`)
   - Install security tools
   - Manage installed tools
   - Delete tools and models

3. **Configuration** (`configuration/`)
   - API key setup
   - Provider configuration
   - System settings

4. **AI Suggestions** (`ai_suggestions/`)
   - OpenAI GPT suggestions
   - Local model suggestions
   - Rule-based autocomplete

5. **Model Management** (`model_management/`)
   - Install AI models
   - Delete models
   - Model status checking

6. **API Management** (`api_management/`)
   - Reload API keys
   - Test connections
   - Status monitoring

## Usage

Each directory contains:
- `README.md` - Feature description and usage
- `config.json` - Configuration template (if applicable)
- `implementation.py` - Core implementation
- `examples/` - Usage examples (if applicable)

## Navigation

Use the main menu in `iblu_assistant.py` to access these options, or navigate directly to the specific directory for detailed documentation.
